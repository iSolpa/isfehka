Panama Electronic Invoicing - HKA Integration

Integración de Facturación Electrónica para Panamá con HKA

This module provides integration with HKA's electronic invoicing system for Panama, enabling compliance with DGI (Dirección General de Ingresos) requirements.

Key Features

  • Send customer invoices and credit notes to HKA for electronic invoicing
  • Automatic CUFE/CAFE generation for fiscal compliance
  • Auto-send on posting for subscriptions and automated invoicing
  • Manual registration of existing HKA documents
  • POS integration for point-of-sale electronic invoicing
  • Cancel and retrieve fiscal documents
  • Company-level configuration for multi-company setups
